# 前端职业素养

## Web 发展史

**Web**（World Wide Web）即全球广域网，也称为**万维网**，它是一种基于超文本和 HTTP 的、全球性的、动态交互的、跨平台的分布式图形信息系统。Web 建立在 Internet 之上，为浏览者提供图形化、易于访问的直观界面，其中的文档及超级链接将 Internet 上的信息节点组织成互为关联的网状结构。

### 起源

1989 年，CERN（欧洲粒子物理研究所）的 Tim Berners-Lee 领导的小组提交了一个针对 Internet 的新协议和文档系统，并将这个新系统命名为 World Wide Web。其目的在于使全球科学家能够通过 Internet 交流工作文档。

1990 年末，该系统的基本框架已在 CERN 的计算机上实现；1991 年正式移植到其他平台并公开发布。

### Web 1.0

Web 1.0 时代始于 1994 年，其主要特征是大量使用**静态 HTML 网页**发布信息，用户通过浏览器单向获取信息。互联网上的资源可以在网页中直观展示，资源之间通过超链接相互关联。Web 1.0 的本质是**聚合、联合、搜索**，解决了人对信息搜索与聚合的需求，但尚未解决人与人之间沟通、互动与参与的问题，由此催生了 Web 2.0。

### Web 2.0

Web 2.0 于 2004 年 3 月由 O'Reilly Media 公司和 MediaLive 国际公司在一次头脑风暴会议上正式提出。Tim O'Reilly 在其文章《What Is Web 2.0》中系统阐述了 Web 2.0 的概念。

Web 2.0 的核心理念是**以用户为中心**，软件作为服务提供，Internet 演化为成熟的网络应用平台，强调用户参与、在线协作、数据的网络化存储与共享。典型应用包括 Blog、Wiki、RSS、SNS、P2P、IM 等，大大激发了互联网的创新活力。

### Web 3.0

Web 3.0 是 Web 2.0 的进一步发展与延伸，其核心是**语义 Web** 与**去中心化**。Web 3.0 通过对微内容进行拆分、词义标准化和结构化，实现信息之间基于语义的链接与互通，使机器能够理解网页内容，从而提供更加个性化、精准化和智能化的检索体验。用户在 Web 3.0 中拥有对自身数据的直接控制权，可以跨平台使用数据，完全基于浏览器即可实现复杂系统的功能。

## 浏览器

浏览器是用来检索、展示以及传递 Web 信息资源的应用程序。Web 信息资源由**统一资源标识符**（Uniform Resource Identifier，URI）标记，可以是网页、图片、视频或其他在 Web 上呈现的内容。用户可借助超级链接（Hyperlinks）通过浏览器浏览互相关联的信息。

### 浏览器内核

浏览器内核分为两个部分：**渲染引擎**（负责解析 HTML/CSS 并渲染页面）和 **JS 引擎**（负责解析和执行 JavaScript）。目前 JS 引擎已趋于独立，"浏览器内核"通常特指渲染引擎。

目前主流的五大浏览器及其内核：

**1. Trident 内核**：代表作品是 IE

因 IE 随 Windows 捆绑安装，曾占有极高市场份额，故又称 IE 内核。此内核仅限 Windows 平台，且非开源。2015 年，IE 浏览器被 Microsoft Edge 取代。目前最新版 Edge 已放弃 Trident，转而使用 Blink 内核。

**2. WebKit 内核**：代表作品是 Safari、早期 Chrome

苹果公司开源的内核项目，被广泛应用于 Android 原生浏览器、Safari 等。Google 在 WebKit 基础上研发了 Blink 内核。

**3. Presto 内核**：代表作品是早期 Opera

该内核于 2003 年在 Opera 7 中首次使用，渲染速度极快，但牺牲了部分网页兼容性。Opera 现已改用 Blink 内核。

**4. Gecko 内核**：代表作品是 Firefox

代码完全开源，跨平台支持 Windows、Linux、macOS 等主流操作系统。

**5. Blink 内核**：由 Google 和 Opera Software 联合开发

Blink 是目前使用最广泛的浏览器渲染引擎，Chrome、Edge、Opera 等主流浏览器均采用此内核。谷歌同时开发了高性能 JS 引擎 **V8**，极大提升了 JavaScript 的执行速度。

## 前端技术栈

前端开发的核心技术由三部分组成：

| 技术 | 职责 |
|------|------|
| **HTML** | 负责网页的**结构**，定义内容和语义 |
| **CSS** | 负责网页的**样式**，控制视觉呈现 |
| **JavaScript** | 负责网页的**行为**，实现交互逻辑 |

在此基础上，现代前端开发通常还涉及：

- **包管理工具**：npm / yarn / pnpm
- **构建工具**：Vite / webpack
- **框架**：Vue / React / Angular
- **版本控制**：Git
- **代码规范**：ESLint、Prettier
- **部署**：Nginx、CI/CD 流水线
