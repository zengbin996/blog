# これは柔らかくて、まるで本物の毛皮のようです

## 基本课文

1. これは柔らかくて、まるでも本物の毛皮のようです。
2. この着物はいかにも日本らしい柄ですね。
   3 .明日の九時までにこの書類を完成させなければなりません。
3. わたしが留学している間に、家の周りもずいぶん変わりました。
