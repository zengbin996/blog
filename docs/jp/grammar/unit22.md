# 森さんは毎晩テレビを見る

## 基本课文

森さんは毎晩テレビを見る。
昨日はとても忙しかった。
コンピュータは簡単ではない。
今日は曇りだ。

明日ボーリングに行かない？
ごめん。明日は仕事があるから。

昨日の試験、どうだった？
ちょっと難しかったけど、まあまあできたよ。

このネクタイ、派手？
ううん、ぜんぜん派手じゃないわ。

森さんの電話番号、知ってる？
うん、知ってる。3493-3945 だよ。
