# 電車が急に止まりました

## 基本课文

1. 電車が急に止まりました。
2. 部屋の電気が消えています。
3. 森さんはボーナスを全部使ってしまいました。
4. このケーキはとてもおいしそうです。

---

- おじいさんにもらった腕時計が壊れてしまいました。
